# Emacs Cheat Sheet

## Install
* Debian/Ubuntu: `sudo apt-get install emacs`

## Cancel
* Abort command: `C-g`
* Undo: `C-/`

## Buffers
* Open: `C-x C-f`
* Kill: `C-x C-k`
* Save: `C-x C-s`
* Save all: `C-x s`
* Switch: `C-x b`
* List all: `C-x C-b`

## Macros
* Start: `C-x (`
* End: `C-x )`
* Execute: `C-x e`

## Clipboard
* Copy: `M-w`
* Kill/Cut: `C-w`
* Yank/Paste: `C-y`

## Mark
* Whole buffer:`C-x h`

## Navigation
* Start of buffer: `M-<`
* End of buffer: `M->`
* Go to line: `M-g g`

## Windows
* Close: `C-x 0`
* Close all: `C-x 1`
* Split vertical: `C-x 2`
* Split horizontal: `C-x 3`
* Switch: `C-x o`

## Help
* Describe key: `C-h k`
