# Vim Cheat Sheet

## Movement

### Cursor
* Left: `h`
* Right: `l`
* Up: `k`
* Down: `j`
* Top: `H`
* Middle: `M`
* Bottom: `L`

### Word
* Previous: `b`
* Next: `w`

### Line
* First: `gg`
* Last: `G`
* Start: `0`
* End: `$`
* Line number 41: `41G`

### Block
* Previous: `}`
* Next: `{`
* To matching delimiter: `%`

### Screen
* Up 1 line: `Ctrl y`
* Down 1 line: `Ctrl e`
* Up 1/2 screen: `Ctrl u`
* Down 1/2 screen: `Ctrl d`
* Up 1 screen: `Ctrl f`
* Down 1 screen: `Ctrl b`
* Center cursor: `zz`
